# 尾牙抽獎遊戲

一套純前端的尾牙/年會抽獎遊戲集合，包含 5 種不同風格的抽獎遊戲，適合公司尾牙、春酒、年會等活動使用。

## 特色

- **純前端**：無需後端伺服器，直接開啟 HTML 即可使用
- **資料持久化**：使用 localStorage 儲存資料，重新整理不會遺失
- **純 CSS 圖形**：所有視覺元素皆由 CSS 繪製，無需額外圖片資源
- **響應式設計**：支援桌面與手機瀏覽器
- **匯出/匯入**：支援 JSON 格式的資料匯出與匯入

## 遊戲列表

### 1. 輪盤抽獎 (Wheel)
經典轉盤抽獎遊戲，使用 Canvas 繪製高解析度輪盤。
- 支援「抽人」與「抽獎品」兩種模式
- 可設定是否允許重複抽取
- 支援復原上一次抽獎

### 2. 抽籤筒 (Picker)
模擬傳統抽籤筒的隨機點名遊戲。
- 籤條從筒中抽出的動畫效果
- 支援黑名單排除功能
- 適合隨機點名或抽獎

### 3. 扭蛋機 (Gacha)
模擬扭蛋機的抽獎遊戲。
- 多色膠囊球代表不同獎項
- 視覺化的扭蛋掉落動畫
- 支援自訂獎項與顏色

### 4. 拉霸機 (Slot)
經典三輪拉霸機遊戲。
- 支援「獎池模式」與「機率模式」
- 獎池模式：設定固定獎項數量，抽完為止
- 機率模式：根據權重隨機抽取，可重複中獎
- 支援「沒中獎」結果設定

### 5. 刮刮樂 (Scratch)
模擬實體刮刮樂的互動遊戲。
- 滑鼠/觸控刮開遮罩層
- 刮開一定比例後自動揭曉結果
- 支援自訂獎項

## 使用方式

### 本機開啟
直接用瀏覽器開啟 `index.html` 即可使用。

### 部署到伺服器
將整個專案資料夾上傳至任何靜態網頁伺服器（如 GitHub Pages、Netlify、Vercel 等）。

## 資料管理

每個遊戲都提供：
- **匯出 JSON**：將目前的設定與抽獎紀錄匯出為 JSON 檔案
- **匯入 JSON**：從 JSON 檔案匯入設定與紀錄
- **重置遊戲**：清除所有資料，恢復初始狀態
